# Segment Me If You Can – Website

This repository contains the source code for the Segment Me If You Can website:
- https://segmentmeifyoucan.com/

![License](https://img.shields.io/badge/license-MIT-green)

### Running the Website Locally
To run the site locally you need [Jekyll](https://jekyllrb.com/) installed.
```bash
git clone https://github.com/SegmentMeIfYouCan/segment-me-if-you-can.git
cd segment-me-if-you-can
bundle install
bundle exec jekyll serve
```
The website will then be available at:
```
http://localhost:4000
```
